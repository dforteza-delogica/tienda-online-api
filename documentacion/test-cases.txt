================================================================================
PRODUCTS
================================================================================

POST /api/products
+------+---------------------------------------------------------------+
| 201  | producto creado correctamente                                 |
| 400  | campos obligatorios vacíos (sku, name, price, stock)          |
| 400  | price <= 0                                                    |
| 400  | stock < 0                                                     |
| 409  | SKU duplicado                                                 |
+------+---------------------------------------------------------------+

GET /api/products
+------+---------------------------------------------------------------+
| 200  | lista de productos (puede estar vacía)                        |
+------+---------------------------------------------------------------+

GET /api/products/{id}
+------+---------------------------------------------------------------+
| 200  | producto encontrado                                           |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

PUT /api/products/{id}
+------+---------------------------------------------------------------+
| 200  | producto actualizado                                          |
| 400  | campos inválidos                                              |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

DELETE /api/products/{id}
+------+---------------------------------------------------------------+
| 204  | eliminado correctamente                                       |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+


================================================================================
CUSTOMERS
================================================================================

POST /api/customers
+------+---------------------------------------------------------------+
| 201  | cliente creado correctamente                                  |
| 400  | campos obligatorios vacíos (fullName, email)                  |
| 400  | email con formato inválido                                    |
| 409  | email duplicado                                               |
+------+---------------------------------------------------------------+

GET /api/customers
+------+---------------------------------------------------------------+
| 200  | lista de clientes (puede estar vacía)                         |
+------+---------------------------------------------------------------+

GET /api/customers/{id}
+------+---------------------------------------------------------------+
| 200  | cliente encontrado                                            |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

PUT /api/customers/{id}
+------+---------------------------------------------------------------+
| 200  | cliente actualizado                                           |
| 400  | campos inválidos                                              |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

DELETE /api/customers/{id}
+------+---------------------------------------------------------------+
| 204  | eliminado correctamente                                       |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

POST /api/customers/{id}/addresses
+------+---------------------------------------------------------------+
| 201  | dirección creada correctamente                                |
| 400  | campos obligatorios vacíos (line1, city, postalCode, country) |
| 404  | cliente no existe                                             |
+------+---------------------------------------------------------------+

PUT /api/customers/{id}/addresses/{addressId}/default
+------+---------------------------------------------------------------+
| 200  | dirección marcada como default                                |
| 404  | cliente no existe                                             |
| 400  | dirección no pertenece al cliente                             |
+------+---------------------------------------------------------------+


================================================================================
ORDERS
================================================================================

POST /api/orders
+------+---------------------------------------------------------------+
| 201  | pedido creado correctamente                                   |
| 400  | campos obligatorios vacíos (customerId, shippingAddressId,    |
|      | items)                                                        |
| 400  | items vacío []                                                |
| 404  | customerId no existe                                          |
| 404  | shippingAddressId no existe                                   |
| 400  | dirección no pertenece al cliente                             |
| 404  | productId no existe                                           |
| 400  | producto inactivo                                             |
| 409  | stock insuficiente                                            |
+------+---------------------------------------------------------------+

GET /api/orders
+------+---------------------------------------------------------------+
| 200  | lista de pedidos (puede estar vacía)                          |
+------+---------------------------------------------------------------+

GET /api/orders/{id}
+------+---------------------------------------------------------------+
| 200  | pedido encontrado con sus items                               |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+

GET /api/orders/customer/{customerId}
+------+---------------------------------------------------------------+
| 200  | lista de pedidos del cliente                                  |
| 404  | cliente no existe                                             |
+------+---------------------------------------------------------------+

PUT /api/orders/{id}/status
+------+---------------------------------------------------------------+
| 200  | CREATED  → PAID        (válida)                               |
| 200  | CREATED  → CANCELLED   (válida)                               |
| 200  | PAID     → SHIPPED     (válida)                               |
| 200  | PAID     → CANCELLED   (válida)                               |
| 400  | SHIPPED  → cualquiera  (inválida)                             |
| 400  | CANCELLED → cualquiera (inválida)                             |
| 400  | CREATED  → SHIPPED     (inválida)                             |
| 400  | PAID     → CREATED     (inválida)                             |
| 404  | id no existe                                                  |
+------+---------------------------------------------------------------+
